## 📌 Power BI Ad Campaign Performance Dashboard

### 📊 Overview

This project is a Power BI dashboard built to analyze and visualize ad campaign performance. It provides insights into campaign effectiveness, user engagement, and conversion metrics to help improve marketing strategies.

---

### 🎯 Objective

* Track overall ad campaign performance
* Identify high-performing and low-performing campaigns
* Analyze CTR, impressions, clicks, and conversions
* Understand cost efficiency and ROI trends
* Support data-driven marketing decisions

---

### 🛠 Tools & Technologies

* Power BI Desktop
* Power Query (Data Cleaning & Transformation)
* DAX (Data Analysis Expressions)
* Excel / CSV Dataset

---

### 📁 Dataset Information

* Dataset Type: Ad Campaign Performance Data
* Size: ~20,000 records
* Key Columns:

  * Campaign Name
  * Impressions
  * Clicks
  * CTR (Click Through Rate)
  * Cost
  * Conversions
  * Date

---

### 📈 Key Insights

* Identified top-performing campaigns based on CTR and conversions
* Found campaigns with high cost but low conversion rate
* Analyzed peak performance time periods
* Compared campaign efficiency using cost vs conversion analysis
* Highlighted underperforming ads for optimization

---

### 📊 Dashboard Features

* KPI Cards (CTR, Clicks, Conversions, Cost)
* Time-based trend analysis
* Campaign-wise performance comparison
* Interactive filters (Date, Campaign, Category)
* Visual insights using bar charts, line charts, and pie charts

---

### 📌 Conclusion

This project demonstrates how Power BI can be used to analyze marketing campaigns and generate actionable insights to improve ad performance and reduce cost inefficiencies.
